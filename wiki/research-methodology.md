## 1. Overview

This page outlines research methodology best practices, with emphasis on
engineering reproducibility, traceability, and formal documentation.

---

## 2. Key Practices

- Define research questions clearly  
- Track assumptions and constraints  
- Use version-controlled experiments (e.g. Git + Jupyter)  
- Document tools, versions, and parameters  
- Ensure reproducibility across simulations  

---

## 3. Suggested Tools

- **Version control:** Git, GitHub  
- **Simulation:** MATLAB, HFSS, CST, Python  
- **Data processing:** NumPy, SciPy, Pandas  
- **Notebooks:** Jupyter, Markdown  

---

## 4. Reproducibility Checklist

- [ ] Simulation source files committed  
- [ ] Parameter sweeps documented  
- [ ] Output post-processed consistently  
- [ ] Data visualizations reproducible
